DECENTRALIZED

Codon is simply a decentralized directory of genetic information. The links are stored using Dat, a peer-to-peer file sharing service built for science. Anyone can host the directory, making it very permanent.

INCENTIVIZED

Users are able to earn rewards for voting on the usefulness of links on the network, as well as for adding usefull links. All rewards will be paid in BASE, an ERC-20 compliant token on the Ethereum network.

USER CURATED

Codon relies on the consesus of the users when determining the most useful and interesting parts. Irrelevent or spam links will be hidden while links showcasing important biological parts stay visible
